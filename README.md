## 🚀 Cómo empezar

1. Clona el repositorio:  
   ```bash
   git clone https://github.com/DevZyxlJs/AlyaBot-MD.git
   ```
2. Instala las dependencias:  
   ```bash
   npm install
   ```
3. Configura tu entorno en el archivo `.env`.
4. Inicia el bot:  
   ```bash
   npm start
   ```
